== Description ==
Wordpress Simplify Mortgage Calculation lets you do the Calculate how much you can afford spending per month on your mortgage/loan payment with this plugin.

== Installation ==

**Normal installation**

1. Download the simplify-mortgage-calculation.zip file to your computer.
1. Unzip the file.
1. Upload the `simplify-mortgage-calculation` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

That's it. You can use the the plugin by using the short code [Mortgage_Calculator] in your page content ro you can call it custom templates php code.
